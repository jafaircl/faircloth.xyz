## Technologies Used

This website is built using [Astro](https://astro.build)

## Running Locally

To run this website locally on your machine, follow these steps:

1. Clone the repository: `git clone https://github.com/jafaircl/faircloth.xyz.git`
2. Navigate into the project directory: `cd faircloth.xyz`
3. Install the dependencies: `npm install`
4. Start the development server: `npm run dev`

The website should now be running at `http://localhost:4321`.
